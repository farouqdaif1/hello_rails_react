# Get Started
To get a local copy up and running follow these simple steps.

```bash
$ git clone https://github.com/farouqdaif1/hello_rails_react.git
$ cd hello-rails-react
$ bundle install 
$ rake db:create 
$ rake db:migrate 
$ rails s 
```
